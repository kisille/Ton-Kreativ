# Sammel-Ordner für neue Fotos

Lade hier einfach **alle Fotos** hoch — mit **beliebigen Namen**, in beliebiger
Reihenfolge. Du musst nichts umbenennen oder sortieren.

**So geht's (am Rechner am einfachsten):**
1. Auf GitHub in diesen Ordner `incoming-photos/` gehen.
2. „Add file → Upload files" → alle Bilder auf einmal reinziehen.
3. „Commit changes".
4. Mir kurz Bescheid geben: „Bilder sind drin."

Dann schaue ich mir jedes Bild an, ordne es dem passenden Platz auf der Website
zu (Porträt, Galerien, Karten …), benenne es korrekt, lege es nach
`public/images/` und räume diesen Ordner wieder auf.

**Wichtig:**
- Bitte als **JPG/PNG** hochladen — **kein HEIC** (iPhone-Standardformat).
  Auf dem iPhone: beim Teilen/Exportieren „JPEG" wählen, oder am Rechner
  umwandeln. HEIC kann ich nicht anzeigen und Browser zeigen es nicht.
- Auflösung egal, je grösser desto besser (ich kann verkleinern).
