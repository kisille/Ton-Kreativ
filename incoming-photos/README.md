# Sammel-Ordner für neue Fotos

Lade hier ein Foto mit **beliebigem Namen** hoch (GitHub: „Add file → Upload
files") und sag mir kurz, **welches Bild es ersetzen soll**. Ich ordne es zu,
benenne es korrekt, lege es nach `public/images/` und räume hier wieder auf.

**Bitte als JPG/PNG** hochladen — **kein HEIC** (iPhone: beim Export „JPEG"
wählen). Auflösung egal, je grösser desto besser.
